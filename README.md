# WH Loader

WH Loader is a lightweight WordPress preloader that uses a Windows 11 style animation. It is designed for the WEBKIH brand to provide a smooth, professional loading transition for websites.

## Features
- Windows-style 4-pane animated loader.
- Customizable brand text (default is WEBKIH).
- Dashboard settings to change background, text, and spinner colors.
- Pure CSS animations for high performance.
- Mobile-friendly and responsive.

## Installation
1. Download the plugin as a .zip file.
2. In your WordPress Dashboard, go to Plugins > Add New.
3. Click Upload Plugin and select the .zip file.
4. Activate the plugin.

## How to Use
Once activated, go to Settings > WH Loader in your WordPress sidebar. From there you can:
- Update the Brand Name text.
- Select a Background Color using the color picker.
- Select a Text Color.
- Select an Active Spinner Color.

## Technical Details
- Built with PHP for the WordPress Plugin API.
- Uses CSS Grid and Keyframes for the animation.
- Includes a small JavaScript snippet to hide the loader once the page is fully loaded.
